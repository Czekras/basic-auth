## Basic Auth

Generate `.htaccess` and `.htpasswd` files for Apache Basic Authentication.

### Dependencies

- [React](https://react.dev) — UI library.
- [@hello-pangea/dnd](https://github.com/hello-pangea/dnd) — Drag and drop for saved accounts and groups.
- [JSZip](https://stuk.github.io/jszip/) — Bundle the generated files into a downloadable zip.
- [lucide-react](https://lucide.dev) — Icon set used throughout the app.

### Dev Dependencies

- [Vite](https://vitejs.dev) — Build tool and dev server.
- [ESLint](https://eslint.org) — Linting, with the React Hooks and React Refresh plugins.
